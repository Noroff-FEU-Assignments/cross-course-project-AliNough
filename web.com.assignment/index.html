Web Security

How would you advise a potential client to secure his/her website from web security attacks? Note: the word count for your answer should not be greater than 250 words

There are diffrent security practices for diffrent type of websites but some common ones are:

Sanitizing of data delivered to the system. Checking every data that try to pass and making sure they use the correct format

Authorization of user who is requesting data or requesting to change or access data. This can be done in form of multistep or multifactore authentication.
